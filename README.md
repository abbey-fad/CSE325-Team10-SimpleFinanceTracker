## **Project Overview**

Simple Finance Tracker is a **.NET Blazor Server application** that helps users manage personal finances.  
Users can:

- Register and log in securely.
- Add income and expense transactions.
- View all transactions in a list.
- View reports summarizing transactions by type.
- View the total balance.
- Log out to secure their account.

The application uses a **clean, responsive design**, adheres to accessibility guidelines (WCAG 2.1, Level AA), and is optimized for a smooth user experience across devices.

---

## **Features**

1. **User Registration & Login**
   - Password masking with a "Show Password" option.
   - Email validation to prevent duplicate accounts.

2. **Transaction Management**
   - Add transactions with date, amount, and description.
   - View a list of all transactions.

3. **Reports & Totals**
   - Summarizes income and expenses by category.
   - Displays total balance.

4. **Responsive UI**
   - Works on desktop, tablet, and mobile.
   - Consistent branding and color scheme.

5. **Logout**
   - Ensures security after the session ends.
